Passage 1: Basic Shortcut Keys

Theory (50 Shortcut Keys with Explanations):

  1. Win + D – Minimizes all windows to show the desktop.
  2. Ctrl + Shift + N – Creates a new folder in File Explorer.
  3. Ctrl + S – Saves the current document in Word or Excel.
  4. Ctrl + C – Copies the selected text or item.
  5. Ctrl + V – Pastes the copied or cut item.
  6. Ctrl + X – Cuts the selected text or item.
  7. Ctrl + Z – Undoes the last action.
  8. Ctrl + Y – Redoes the previously undone action.
  9. Ctrl + B – Applies bold formatting in Word.
  10. Ctrl + I – Applies italic formatting in Word.
  11. Ctrl + U – Underlines selected text in Word.
  12. Win + E – Opens File Explorer.
  13. Win + L – Locks your Windows computer.
  14. Alt + Tab – Switches between open applications.
  15. Win + R – Opens the Run dialog box.
  16. Win + I – Opens Windows Settings.
  17. Win + A – Opens the Action Center.
  18. Win + S – Opens the Windows Search bar.
  19. Win + X – Opens the Quick Link menu.
  20. Win + M – Minimizes all open windows.
  21. Win + Shift + M – Restores minimized windows.
  22. Ctrl + A – Selects all content in a document.
  23. Ctrl + F – Opens the Find dialog.
  24. Ctrl + H – Opens the Replace dialog in Word.
  25. Ctrl + P – Opens the Print dialog.
  26. Ctrl + N – Creates a new document in Word.
  27. Ctrl + O – Opens an existing document.
  28. Ctrl + K – Inserts a hyperlink in Word.
  29. Ctrl + D – Opens the Font dialog in Word.
  30. Ctrl + E – Centers text in a document.
  31. Ctrl + L – Left-aligns text.
  32. Ctrl + R – Right-aligns text.
  33. Ctrl + J – Justifies text in Word.
  34. Ctrl + M – Increases the paragraph indent in Word.
  35. Ctrl + Shift + M – Decreases the paragraph indent.
  36. F7 – Runs the Spelling and Grammar check in Word.
  37. Shift + F7 – Opens the Thesaurus in Word.
  38. Ctrl + Shift + C – Copies the formatting (Format Painter) in Word.
  39. Ctrl + Shift + V – Pastes the copied formatting in Word.
  40. Ctrl + Enter – Inserts a page break in Word.
  41. F2 – Edits the active cell in Excel.
  42. F11 – Creates a chart based on selected data in Excel.
  43. Ctrl + ` – Toggles the display of formulas in Excel.
  44. Ctrl + ; – Inserts the current date in Excel.
  45. Ctrl + Shift + : – Inserts the current time in Excel.
  46. Alt + F4 – Closes the current window or application.
  47. Win + Tab – Opens Task View to show all desktops.
  48. Win + Ctrl + D – Creates a new virtual desktop.
  49. Win + Ctrl + F4 – Closes the current virtual desktop.
  50. Win + Plus (+) – Opens the Magnifier tool and zooms in.
